# Superstore Sales Analysis

## Overview
This project analyzes Superstore sales data to identify sales trends, profit performance, and business insights.

## Objectives
- Analyze sales and profit trends
- Identify top-performing categories
- Compare regional performance
- Generate business insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Sales trend analysis
- Profit analysis
- Category-wise performance
- Regional comparison

- Dataset
- Analysis Notebook
- Visualizations
